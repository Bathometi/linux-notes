# Шпаргалка з Linux для OSINT

## 1. Базовий конвеєр для аналізу даних
Збиває дані в купу, рахує кількість унікальних записів і сортує від більшого до меншого:
`sort | uniq -c | sort -nr`

## 2. Команда cut (нарізка стовпчиків)
Вирізати пошти (3-й стовпчик, роздільник двокрапка):
`cut -d ":" -f 3 users_db.txt`

Вирізати нікнейми з посилань (2-й стовпчик, роздільник коса риска):
`cut -d "/" -f 2 social_profiles.txt`

Вирізати країни (3-й стовпчик, роздільник знак рівності):
`cut -d "=" -f 3 companies_audits.txt`

## 3. Повний конвеєр з обрізанням топу (head)
Витягти гаманці, порахувати унікальні, відсортувати топ і показати тільки 1 перший рядок:
`cut -d "," -f 2 crypto_trans.txt | sort | uniq -c | sort -nr | head -n 1`

*Важливо: назву файлу пишемо тільки один раз на самому початку конвеєра!*


### 3. Потужна фільтрація та вивід через AWK ✂️

Коли `cut` вже не справляется, ми використовуємо `awk`. Він автоматично розуміє пробіли як роздільники й дозволяє міняти стовпчики місцями або додавати текст.

*   `$1`, `$2`, `$3` — номери стовпчиків (полів).
*   `$0` — весь рядок повністю.

#### Базовий вивід (роздільник за замовчуванням — пробіл):
```bash
# Поміняти місцями 2-й та 1-й стовпчики
awk '{print $2, $1}' suspects.txt

# Якщо дані розділені двокрапкою ":"
awk -F ":" '{print "Користувач: " $1}' users.txt

# Вивести тільки телефон ($2) тих, хто з міста Kyiv ($3)
awk -F ";" '$3 == "Kyiv" {print "Номер телефону: " $2}' phones.txt
